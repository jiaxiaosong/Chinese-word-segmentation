中文分词程序
本程序主要功能为对中文文本进行分词和分句
作者：贾萧松 柴峥豪 李明泽（Three's Company）
开发环境：python3.5+tkinter+pycharm
系统平台：win10
------------------------------------------------------------------------------------
使用说明
1、本程序支持txt文本导入(-utf8 编码)和键盘输入文本。
2、本程序可以输出程序内文本，格式txt。
3、本程序有两种分词算法（正向逆向），用户可自行选择
